Developed a performance monitoring tool during the testing phase to evaluate key network performance indicators such as latency, jitter, packet loss, and TCP/UDP throughput for LEO satellite communication networks across multiple cities. I assisted in setting up the testing environment, configuring probe devices, executing performance tests, and analyzing collected data to assess link behavior and identify performance variations across different deployment regions.
The collected KPI data was visualized using Grafana dashboards to enable real-time monitoring and comparative performance analysis across locations.
